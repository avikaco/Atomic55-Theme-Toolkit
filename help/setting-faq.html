Frequently Asked Questions

How I get custom theme customizer value?
You can use a55_option('FIELD NAME') function, example:
a55_option('contact_phone');

Can I hide theme setting (this page) from my user?
Yes you can, add this code to your functions.php file.
define('ATOMIC55_DISABLE_THEME_SETTING_PAGE', TRUE); // if you want to re-enable this page, change to FALSE

Can display map from my theme setting?
Yes you can, There is 2 ways to display map from your theme setting

  1. Using PHP function:
    a55_option_map ($control_id, $use_static_map = TRUE, $width = '100%', $height = '400px');
  2. Using Wordpress Shortcode:
    [a55_option_map controlid="your_map_control_name" static="1" width="100%" height="400px"/]

Can I create predefined section/control?
Yes you can, add this code to your functions.php file.
a55_theme_section_register($section_id, $args = array());
a55_theme_control_register($section_id, $control_id, $attr = array());
